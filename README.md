# 🏛️ GUANDA-EC Organization Meta Repository

這是 **GUANDA-EC (廣達電控)** 組織的元數據儲存庫 (Meta Repository)。
這個儲存庫的主要功能是管理組織層級的設定、公共模板以及展示在組織首頁的門面內容。

---

## 🛠️ 此儲存庫的功能

### 1. 組織門面 (Profile README)
* **路徑：** `profile/README.md`
* **功能：** 這是顯示在 [GUANDA-EC 組織首頁](https://github.com/GUANDA-EC) 的主要介紹頁面。
* **修改說明：** 若要更新組織的簡介、技術棧標籤 (Badges) 或專案連結，請編輯此路徑下的檔案。

### 2. 全域預設檔案 (Global Defaults)
未來若在此儲存庫的根目錄建立 `.github/` 資料夾，可以放置：
* **ISSUE_TEMPLATE/**: 定義組織內所有專案通用的 Issue 格式。
* **PULL_REQUEST_TEMPLATE.md**: 定義全域的 PR 審核清單。
* **CONTRIBUTING.md**: 說明外部或內部成員如何參與本組織專案的規範。

---

## 🚀 如何維護與更新

### 更新組織首頁
1.  進入 `profile/README.md`。
2.  使用 Markdown 編輯內容。
3.  **注意：** 建議保持簡潔，並確保圖片 (如 Banner 或 Stats 圖表) 的連結有效。

### 常用工具連結
* [Shields.io](https://shields.io/): 用於製作技術棧標籤。
* [GitHub Readme Stats](https://github.com/anuraghazra/github-readme-stats): 用於更新動態統計圖表。
